# PROJECT — FinLedger

**FinLedger** é um aplicativo desktop de finanças pessoais cujo objetivo é permitir que o usuário registre receitas e despesas, visualize seu saldo mensal e histórico, organize gastos por categorias e importe dados de planilhas externas.

O foco inicial (V0) é estabelecer a versão de paridade do protótipo com um backend Tauri persistindo os dados em um banco SQLite local, permitindo acesso offline seguro.
